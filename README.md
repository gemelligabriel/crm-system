# Customer Relationship Management - AGES

A full-stack MVP for simple patient management.

## Structure

```txt
backend/   REST API built with NestJS, Prisma, and SQLite
frontend/  React app with TypeScript, React Router, and Axios
````

## How to run

### Backend

```bash
cd backend
npm install
npm run prisma:generate
npm run prisma:migrate
npm run start:dev
```

The API runs at `http://localhost:3000`.

### Frontend

In another terminal:

```bash
cd frontend
npm install
npm run dev
```

The app runs at `http://localhost:5173`.
